# Session 01: Introduction to React

## 💪 Motivation

This sessions will guide you through the very basics of the JavaScript library React. 

You'll learn to apply your knowledge on HTML and CSS in a React codebase. 

## 🎯 Learning objectives

- Knowing basic concepts of React
- Understanding React components and composition
- Creating React projects with `vite`
- Knowing differences between JSX and HTML
- Applying styles to React components
- Applying global styles to a React application 
- Using assets in React projects
